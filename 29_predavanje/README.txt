Za korištenje ove datoteke potrebno je napraviti sljedeće korake u Terminalu:
    1. npm init --> inicijalizacija Node.js 
        - sve korake preskočiti osim entry point gdje je potrebno dodati app.js
    2. npm install express - instalacija Express-a
    3. npm install mongoose -- instalacija mongoose biblioteke za rad s MongoDb bazama podataka

Aplikacija se pokreće iz terminala naredbom: node app
Aplikaciju je moguće testirati primjerice pomoću Postman-a
